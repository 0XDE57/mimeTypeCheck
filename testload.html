MIME type check

If you get a javascript alert, your browser did not enforce MIME type check before executing the .gif


Check console for error message:

Chrome: Refused to execute script from '.../test.gif' because its MIME type ('image/gif') is not executable.

Firefox: Loading script from file: URI (“.../test.gif”) was blocked because its MIME type (“image/gif”) is not a valid JavaScript MIME type.

To fix on Firefox, set the about:config property security.block_fileuri_script_with_wrong_mime = true
